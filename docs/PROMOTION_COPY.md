# NotePDF2PPT 推广文案包

## 1) 一句话介绍（Slogan）

- 把 PDF 变成真正可编辑的 PPTX，而不只是截图拼接。

## 2) 电梯稿（30 秒）

NotePDF2PPT 是一款面向内容创作者、讲师和咨询从业者的 PDF 转 PPTX 工具。它会先用 AI 清理页面底图中的原文字，再重建可编辑文本框，尽量还原原稿风格。相比传统转换器，结果更接近“可继续修改的源文件”，适合快速改稿、复用和演示。

## 3) 详情页介绍（可放 GitHub 首页）

你是否遇到过这些问题：

- PDF 转 PPT 后全是图片，无法编辑
- 文本位置错位、字号混乱、改起来比重做还慢
- 有水印或底图文字，改完仍然重影

NotePDF2PPT 的方案是：

1. OCR + 语义分析识别文本内容与布局
2. AI 对底图进行去字修复
3. 回填可编辑文本框并生成 `.pptx`

目标不是“看起来像”，而是“能继续改”。

## 5) 使用注意点（对外说明建议原样保留）

1. 建议不要开 VPN，避免网络链路不稳定影响上传与渲染成功率。
2. 本项目采用 NanoBanana-pro 云端渲染，会产生算力费用（建议在产品内明确计费规则）。
3. AI 生成存在一定随机性（“抽卡”现象），同一输入可能出现细节差异。
4. 生成的 PPTX 在复杂场景下可能需要少量人工微调，这是当前 AI 转换的正常边界。
