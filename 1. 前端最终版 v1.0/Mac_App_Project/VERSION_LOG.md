## 如何回退版本

### 方法1：告诉 Claude
直接说 **"回退到 v0"** 或 **"恢复到版本 v0"**，Claude 会帮您执行回退。

### 方法2：自己操作 Git 命令

```bash
# 查看所有版本
git log --oneline

# 回退到特定版本（保留修改但不提交）
git checkout <commit-hash> .

# 完全回退（丢弃之后的所有修改）
git reset --hard <commit-hash>
```

## 版本详情

### v2.7 - 注册功能回归与体验优化 (2026-02-13)
**核心变更**：
- **恢复用户注册**：新手机号登录自动注册，无需管理员手动添加。
- **新人福利**：新用户注册即送 5 积分。
- **UI 纯净版**：移除 Mac App 和 Web 后台的“密码登录”选项，仅保留“验证码登录”。
- **界面优化**：Mac App 首页文案调整，移除多余说明文字；积分同步机制修复（解决刷新闪烁问题）。

## 版本详情

### v1 - 修复字体大小不一致 (2026-01-17)
**问题**：PDF 转 PPTX 后，文字大小与原 PDF 不匹配

**修改内容**：
- `OCREngine.swift`: 简化 fontSizeRatio 计算，直接使用 OBB 高度 × 0.92
- `PPTXGenerator.swift`: 添加标准字号吸附（snapFontSize），确保字号一致性
